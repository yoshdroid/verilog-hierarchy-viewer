# Verilog Hierarchy Viewer

VS Code extension project for inspecting Verilog HDL/SystemVerilog module hierarchy.

## Goal

Given Verilog HDL/SystemVerilog source files in a workspace, the extension lets the user choose a top module, shows the instantiated hierarchy below it, and jumps from each instance node to the corresponding source line.
